# 📖 Engineering Journey

My journey into software engineering began unexpectedly.

In **2016**, I co-founded **Gaur Brothers**, a YouTube channel focused on anime and cartoons. As our audience grew, I wanted a platform beyond YouTube where we could publish articles, news, and original content. That idea led to the creation of **Toon World**.

Building Toon World introduced me to HTML, CSS, JavaScript, website customization, and SEO. What started as curiosity gradually became a passion for software engineering.

Today, I build full-stack applications, cloud systems, and AI-powered products with the same mindset I've always had:
